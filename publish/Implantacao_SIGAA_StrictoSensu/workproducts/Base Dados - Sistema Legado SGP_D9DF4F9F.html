Description
Main DescriptionEste produto de trabalho representa a totalidade dos dados extraídos do sistema acadêmico legado (SGP). Inclui dados de alunos, docentes, matrículas, histórico escolar, informações financeiras e outras categorias relevantes para o funcionamento do SIGAA. Sua estrutura pode variar conforme o formato da extração (ex: arquivos CSV, XML, dumps de banco de dados).
Brief OutlineOs dados são obtidos através de exportação direta do banco de dados do SGP ou de relatórios/extrações específicas.
Notation

Selected RepresentationArquivos de texto delimitados (CSV), planilhas eletrônicas, dumps de banco de dados, ou formato XML.
Key Considerations
A qualidade e a integridade dos dados de origem são críticas para o sucesso da migração. Deve-se garantir a segurança e a confidencialidade dos dados durante a extração e o armazenamento temporário. A volumetria dos dados pode impactar o tempo e os recursos necessários para a migração.
Tailoring
Impact of not havingA ausência desses dados impede a continuidade do histórico e a funcionalidade completa do SIGAA.
Reasons for not needingNão aplicável, pois é um artefato essencial para a implantação.
