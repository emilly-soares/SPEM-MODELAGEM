Artifact: Relatório de Plano de Migração de Dados
Documento que descreve a estratégia, escopo, cronograma, ferramentas e procedimentos para a migração de dados.
Purpose
Orientar e controlar todas as atividades envolvidas na migração dos dados do sistema SGP para o SIGAA, minimizando riscos e garantindo a integridade dos dados.

Description
Main DescriptionEste plano detalha as etapas, responsabilidades, pré-requisitos, ferramentas, procedimentos de extração, transformação e carga (ETL), estratégias de validação, planos de contingência e rollback, cronograma e recursos necessários para a migração dos dados do sistema legado para o novo sistema.
Brief OutlineO plano é elaborado na fase inicial do projeto de migração e é revisado e aprovado pelas partes interessadas.
NotationFormato de documento textual, diagramas de fluxo (BPMN ou similar).
Selected RepresentationDocumento de texto (Word, PDF), ou diagrama de fluxo de processos.
Key Considerations
Deve contemplar a complexidade dos dados e a volumetria. Incluir planos de teste e validação rigorosos. Definir claramente os papéis e responsabilidades de cada envolvido. Considerar janelas de manutenção e impactos operacionais.
Tailoring
Impact of not havingA ausência do plano pode levar a migrações desorganizadas, erros, perda de dados e atrasos no projeto.
Reasons for not needing

Não aplicável, pois é um artefato crítico para a segurança e sucesso da migração de dados.
