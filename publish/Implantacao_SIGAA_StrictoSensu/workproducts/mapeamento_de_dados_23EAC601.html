Artifact: Mapeamento de dados
Documento que detalha a correspondência entre os campos e estruturas de dados do sistema SGP e do SIGAA.
Purpose
Garantir que os dados sejam corretamente transferidos e transformados do sistema legado para o novo sistema, mantendo a integridade e a semântica da informação.

Description
Main DescriptionEste produto de trabalho é um documento técnico que define como cada elemento de dado no sistema de origem (SGP) se relaciona com o correspondente elemento no sistema de destino (SIGAA). Inclui informações sobre tipo de dado, formato, regras de transformação, chaves primárias e estrangeiras, e quaisquer exceções ou considerações especiais para cada campo.
Brief OutlineO mapeamento é desenvolvido através da análise dos esquemas de banco de dados e modelos de dados de ambos os sistemas.
NotationTabelas comparativas; Diagramas de mapeamento de dados (se aplicável).
Selected RepresentationDocumento de texto (Word, PDF), planilha eletrônica (Excel), ou ferramenta de modelagem de dados (UML, ERD).
Key Considerations
A precisão do mapeamento é fundamental para evitar perda ou corrupção de dados. Deve ser validado por especialistas do negócio e técnicos de ambos os sistemas. As regras de transformação devem ser claramente documentadas e implementadas.
Tailoring
Reasons for not needingSem o mapeamento, a migração de dados seria inconsistente e resultaria em erros e perda de informações.
