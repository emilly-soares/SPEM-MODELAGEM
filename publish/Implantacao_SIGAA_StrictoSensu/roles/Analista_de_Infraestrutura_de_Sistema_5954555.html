    Main Description
    O Analista de Infraestrutura de Sistemas é o profissional responsável pelas atualizações de versão do sistema, por realizar os merges necessários no repositório de código e pela disponibilização do sistema nos ambientes de teste, homologação e produção. Este papel está alocado principalmente na DDS, embora, quando necessário, analistas vinculados à DS também possam ser acionados para apoiar nas atividades.
    Staffing
    SkillsAdministração de sistemas; Gerenciamento de repositórios de código (Git/SVN); Automação de deploy; Gerenciamento de ambientes (teste, homologação, produção); Conhecimento em infraestrutura de TI.
