    Main Description
    Profissionais que estão alocados nas  estruturas organizacionais que irão realizar o gerenciamento da nova ferramenta (DDPS, DGP, PROPEPG),estes usuários serão referencia para os usuários finais do novo sistema (docentes e TAEs dos campi da UFFS),  são responsáveis por testar o sistema com o intuito de compreender o funcionamento da nova ferramenta e com isso poderem adequar os fluxos dos processos, também são responsáveis pela validação das etapas de migração de dados.
    Staffing
    SkillsConhecimento dos fluxos de trabalho das unidades (DDPS, DGP, PROPEPG); Testes de software (funcionais); Validação de dados; Comunicação de feedback.
