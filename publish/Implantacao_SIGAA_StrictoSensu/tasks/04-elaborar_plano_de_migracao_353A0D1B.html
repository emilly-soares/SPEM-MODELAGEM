Task: Elaborar Plano de Migração
Etapa responsável por criar um documento descrevendo a ordem de migração dos dados do sistema legado.

      Outputs
      • Documentação de Implantação
      • Relatório de Plano de Migração de Dados
        Process Usage
        • Processo de implantação  >  Execução Equipe de TI  >  Elaborar Plano de Migração
        Main Description
        Tarefa responsável por criar um documento descrevendo as etapas/entregas planejadas pela equipe de implantação para a migração dos dados  e também a ordem em que os dados serão migrados. Este documento abordará também o escopo dos dados que serão ou não serão migrados
        Steps
        Analisar base de dados do SIGAA
        Analisar a base de dados do SGP
        Criar relação dos dados que serão ou não migrados
        Definir ordem das tabelas a serem migradas
        Criar relatório de Plano de Migração

        migração, etapas
