Task: Migrar Dados
Transferência e transformação de dados do sistema legado (SGP) para o novo sistema (SIGAA).
Purpose
Garantir a integridade e disponibilidade dos dados históricos e operacionais, transferindo-os do sistema SGP para o SIGAA, de forma a permitir a continuidade das operações no novo ambiente.

      Process Usage
      • Processo de implantação  >  Execução  >  Migrar Dados
      Main Description
      Esta tarefa envolve a extração de dados do sistema SGP, a sua transformação para o formato compatível com o SIGAA, e a carga final para o banco de dados do novo sistema. Inclui a validação da consistência e completude dos dados migrados.
      Steps
      Realizar Analise para montar o mapeamento da migração
      Montar o mapeamento
      Executar a migração
      Validação pelo Analista de Sistemas
      Key Considerations
      A migração deve ser realizada em ambiente controlado. É crucial planejar o tempo de inatividade necessário para a migração. Deve-se garantir a segurança dos dados durante todo o processo.
