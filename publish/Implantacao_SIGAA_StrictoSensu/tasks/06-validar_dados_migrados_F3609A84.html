      Process Usage
      • Processo de implantação  >  Execução Equipe de TI  >  Validar Dados Migrados
      Main Description
      Etapa responsável por validar os  dados que foram migrados em uma determinada etapa e que foram entregues para a equipe de testes realizar as análises necessárias. Esta etapa irá ocorrer diversas vezes de acordo com o planejado pelo documento Plano de Migração de Dados.
      Steps
      Receber demanda do Analista de Sistemas
      Analisar documentação recebida
      Testador deve acessar o sistema

      Pré-Requisitos:
      O sistema deve estar em ambiente de homologação ou testes,com base de dados atualizada
      O usuário/testador deve possuir credenciais de acesso válidas
      O usuário/testador deve possuir nível de acesso liberado para a interface desejada
      O usuário/testador deve possuir permissão para realizar consultas na base de dados

      Validar os dados migrados
      Elaborar relatório de validação da migração
      Key Considerations
      O Analista de teste, deve conferir os dados migrados tanto diretamente na base de dados, quanto conferir nas interfaces para verificar se os dados foram migrados de fato para as tabelas corretas.
