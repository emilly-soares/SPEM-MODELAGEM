      Main Description
      Nesta etapa os usuários devem acessar o sistema e realizar os testes na aplicação, afim de conhecer como o sistema funciona em cada funcionalidade.
      Steps
      Usuário acessa o sistema

      Pré-Requisitos:
      O sistema deve estar em ambiente de homologação ou testes,com base de dados atualizada
      O usuário deve possuir credenciais de acesso válidas
      O usuário deve possuir nível de acesso liberado para a interface desejada


      Usuário acessa a interface desejada
      Usuário informa os dados para teste
      Usuário aciona a ação desejada no sistema
