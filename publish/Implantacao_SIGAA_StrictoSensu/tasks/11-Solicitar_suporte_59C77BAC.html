Task: Solicitar Suporte
Purpose
Permitir que os usuários reportem problemas, tirem dúvidas ou solicitem auxílio técnico para garantir o uso contínuo e eficiente do sistema.

    • Processo de implantação  >  Execução  >  Solicitar Suporte
    Main Description
    Esta tarefa descreve os passos que um usuário deve seguir para solicitar suporte. Isso inclui a identificação do problema, a escolha do canal de comunicação (ex: sistema de chamados, e-mail, telefone) e o fornecimento das informações necessárias para que o suporte possa ser prestado de forma eficaz.
    Steps
    Usuario acessa sistema de chamados
    Usuario cadastra a solicitação
    Key Considerations
    A clareza na descrição do problema é fundamental para agilizar o atendimento. Definir canais de comunicação de suporte acessíveis. Incentivar o usuário a verificar a documentação antes de solicitar suporte.
