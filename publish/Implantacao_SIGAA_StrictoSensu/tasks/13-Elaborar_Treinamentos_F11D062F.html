    Main Description
    Processo para o usuário criar a documentação que será utilizada nos momentos de treinamentos com os usuários finais do sistema. Esta documentação deverá ser gerada para cada nivel de acesso diferente que os usuário terão. Nessa etapa também deverá ser escolhido se sera utilizado uma base de dados com dados reais ou embaralhados nas etapas de treinamento.
