      Process Usage
      • Processo de implantação  >  Homologação  >  Treinar Usuários
      Main Description

      Etapa onde os usuários da PROPEPG com o auxílio da DS, repassam os treinamentos necessários para os demais servidores da universidade que irão utilizar este sistema. Esta tarefa pode ser executa multiplas vezes, pois cada treinamento é para um nível de acesso especifico no sistema. (Docentes, Coordenadores, Alunos, Secretarias de Curso, Secretaria Acadêmica)


      Steps
      Agendar Treinamento
      Comunicar usuários interessados
      Definir Forma do Treinamento(Presencial ou Remoto)
      Reservar Sala
      Repassar Treinamento
      Gerar Relatório da Execução
