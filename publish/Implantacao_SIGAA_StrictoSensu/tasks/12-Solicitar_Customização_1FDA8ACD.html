Task: Solicitar Customização
Cadastrar solicitação de customização para a equipe de implantação
Purpose
Permitir que os usuários reportem interfaces que precisam ser customizadas para garantir o uso contínuo e eficiente do sistema.
Relationships
RolesPrimary Performer:
  • Usuários PROPEPG
Additional Performers:
  • Analista de Negócios
  • Analista de Sistemas
  • Analista de Suporte
InputsMandatory:
  • Base Dados Atualizada - Novo Sistema (SIGAA)
  • Sistema Disponível (Ambientes de Desenvolvimento, Testes e Homologação)
    		Optional:
    • None
    Outputs
    • Chamado - Solicitação de Customização
    • Documentação de Implantação
      Process Usage
      • Processo de implantação  >  Homologação  >  Solicitar Customização
      Main Description
      Etapa onde os usuários podem solicitar ajustes em funcionalidades do sistema. Esta solicitação sera analisada pela equipe de implantação.
      Steps
      Usuário acessa o sistema de chamados
      Usuário realiza o cadastro de solicitação
